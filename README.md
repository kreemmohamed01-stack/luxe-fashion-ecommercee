# LUXE – Luxury Fashion E-Commerce

A premium luxury fashion e-commerce website built with a modern UI, smooth animations, and a responsive user experience.

## ✨ Features

- Luxury UI Design
- Responsive Layout
- Cinematic Hero Section
- Smooth GSAP Animations
- Product Search
- Wishlist
- Shopping Cart
- Checkout Experience
- Elegant Hover Effects

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- GSAP
- AOS
